# AI CSV Query System

This project uses an LLM to query CSV data using natural language.

## Features
- Converts questions into pandas queries
- Executes generated code
- Handles structured data intelligently

## Tech
Python, Pandas, LangChain

## Note
API keys are removed for security.

## How to Run

1. Install dependencies:
pip install pandas langchain langchain-openai

2. Update API configuration in the code

3. Run:
python AI_CSV_QUERY.py
